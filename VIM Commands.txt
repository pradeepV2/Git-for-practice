1. Command for entering into EDIT Mode
-> vim file123.txt

2. Activate the EDIT Mode-> 
Type 'I'

3. Use Arrow keys for navigation

4. Type the data and Use Delete and Backspace for deleting the data

5. Save the File and Exit from the Edit Mode
-> Esc :wq!
-> Esc :wq

6. If you dont want to save the changes
Esc :q!

7. This will not let you to Exit the file, It will ask you to press !
Esc :q
